===============================
bambu
===============================

Bambu integrates Pandas and RDF

* Free software: BSD license

.. * Documentation: https://bambu.readthedocs.org.

Features
--------

* [ ] ``to_rdf``
* [ ] ``read_rdf``

* [ ] ``to_rdfa``
* [ ] ``read_rdfa``

* [ ] ``to_jsonld``
* [ ] ``read_jsonld``

See: https://github.com/pydata/pandas/issues/3402


Installation
-------------

1. Install pandas

2. Install bambu::

    pip install -e https://github.com/westurner/bambu#egg=bambu
